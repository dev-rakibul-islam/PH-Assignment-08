### App Name

AppDeck

### Description

AppDeck helps you discover, compare, and install the right apps for your work and life. Browse curated lists, check ratings, and manage your installs.

### Technologies

- React 19
- Vite 7
- React Router DOM 7
- Tailwind CSS 4
- DaisyUI
- Recharts
- React Icons
- React Toastify
- ESLint
